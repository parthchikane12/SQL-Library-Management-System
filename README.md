# 📚 Library Management System – SQL Project

This project demonstrates a **simple Library Management System database** built using SQL.
It showcases fundamental database concepts such as **table creation, relationships, data insertion, and SQL queries**.

The system manages information related to **books, authors, library members, and borrowing records**.

---

## 🚀 Features

* Database creation using SQL
* Table relationships using **Primary Key and Foreign Key**
* Insert sample records into tables
* Retrieve data using **SELECT queries**
* Perform **JOIN operations**
* Example queries for data analysis

---

## 🗂 Database Structure

The project contains the following tables:

1. **Authors** – Stores author information
2. **Books** – Stores book details and author reference
3. **Members** – Stores library member information
4. **Borrow** – Tracks borrowed books and return dates
